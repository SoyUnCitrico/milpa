Revolution III

por mdrn.mx

Pieza de timeline (≈30 s) por millis(): un atardecer de círculos concéntricos da paso a un rayo de luz y a un relieve trazado con ruido, y se detiene con noLoop(). Composición vertical.

Controles

  • sGuardar el canvas como PNG (saveCanvas).

80function relieve(tiempo, radioInt, radioExt, ruido) {
81    push();
82    translate(w/2, h*2/3);
83    strokeWeight(1);
84
85    let x = radioExt * cos(TWO_PI * tiempo);
86    let y = radioExt * -sin(TWO_PI * tiempo);
87    let a = radioInt * cos(TWO_PI * tiempo);
88    let b = - radioInt * noise(ruido) * -sin(TWO_PI * tiempo); 
89    console.log(b);
