Girasol

por mdrn.mx

Girasol estático compuesto por cuatro espirales de partículas (centro, dos coronas de semillas y pétalos) sobre un cielo azul. Estudio de composición polar; reusa la biblioteca Spiral y se detiene con noLoop() tras el primer frame.

113function setSpiral(ciclos = 1, radioInt = 10, radioExt = 100, stepsRadio = 200, totalParticles = 100, particleSize = 5, particleStroke = 2, particleType = 'circle', particleColor = '#000000') {
114  const spiral = [];
115  for(var i = 0; i < totalParticles; i++) {
116      let angleIncrement = 360 / totalParticles * ciclos;
117      let angle = TWO_PI / 360 * (angleIncrement * i);
118      let actualRadio = ((radioExt - radioInt) * i / stepsRadio) + radioInt;
119      let xPos = actualRadio * sin(angle) + width / 2;
120      let yPos = actualRadio * cos(angle) + height / 2; 
121      const newPos = createVector(xPos, yPos);
122      const newParticle = new Particle(newPos, particleSize, particleStroke, particleType, particleColor, angle);
123      spiral.push(newParticle);
124  }
125  return spiral;
126}
